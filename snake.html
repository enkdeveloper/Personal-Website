Instructions

Use arrow keys to control the snake:

  • ( ↑ ) Move Up
  • ( ↓ ) Move Down
  • (←) Move Left
  • (→) Move Right

History

Snake Game is a classic arcade game first introduced in the late 1970s. It gained widespread popularity on early Nokia mobile phones in the late 1990s.
