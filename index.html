About Me

Hello, I'm Elif Naz Karakız. I graduated from the Dentistry Faculty in 2021 and actively worked as a dentist in Istanbul for 2 years. During this period, I embarked on a journey that started with my interest in software development and evolved into a passion.

In the past year, I've searched into modern technologies such as CSS, HTML, JavaScript, React JS, and Node JS, starting with the basics and gradually moving on to more complex projects. This ongoing journey has been both challenging and rewarding. My professional education in this field has equipped me to keep pace with the dynamic requirements of the software world.

Throughout this odyssey, I've crafted various projects using programming languages like Java, C, and Python. Sharing these endeavors on GitHub and contributing to the open-source community has been a source of both professional and personal fulfillment. Feel free to explore my LinkedIn profile and search through my projects on GitHub.

On my website, you'll find an array of projects showcasing my skills and passion for software development. The first page serves as an introduction and includes this overview. The subsequent pages feature a Snake Game, a Memory Game, and a detailed Roadmap for aspiring Front-end Developers. In the Roadmap, I share my personal experiences, the technologies I've used, and the paths I've taken to become a front-end developer. Feel free to explore each section to gain insights into my journey and expertise in the world of web development.
